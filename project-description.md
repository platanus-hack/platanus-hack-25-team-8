# Filtro de Llamadas Seguro

Este proyecto es un sistema de **protección contra estafas telefónicas** que filtra y anonimiza información sensible, protege la identidad de voz y alerta al usuario sobre posibles riesgos.

El proyecto incluye:

- **Frontend:** Interfaz web en React para simular y gestionar llamadas.
- **Backend:** API Python FastAPI que gestiona llamadas, filtros y notificaciones.

Deploy:

- **Frontend:** "https://frontend-anonym-audio.vercel.app/"
- **Backend:** "backendanonymaudio-production.up.railway.app"

## Funcionalidades

- **Preventivo:** Bloquea llamadas sospechosas antes de que lleguen al usuario.
- **Reactivo:** Permite acciones sobre llamadas ya recibidas según riesgo detectado.
- **Anónimo & Privado:** Protege la privacidad del usuario y anonimiza la identidad de la voz.
- **Alertas accionables:** Notificaciones en tiempo real que permiten actuar ante riesgo.
- **Contactos:** Gestión de contactos confiables y whitelist.
- **Alarmas y Notificaciones:** Informan sobre intentos de fraude o llamadas sospechosas.
- **Protección de identidad de voz:** Cambia la voz original para proteger al usuario.
- **Cambio de voz original:** Modifica la voz en tiempo real durante la llamada.
- **Filtro de info sensible:** Detecta y oculta datos privados de audio.
- **En tiempo real:** Todas las funciones actúan mientras la llamada ocurre.
- **Cancela llamadas si estima riesgo:** Bloquea automáticamente llamadas que parezcan fraude.
